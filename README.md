# Hertz-Prime 的核桃助手

Windows 平台的 Warframe 开核桃/虚空遗物辅助工具。通过自动 OCR 或全局快捷键截取遗物奖励画面，调用系统 OCR 识别物品名称，查询 Warframe Market 实时卖价，并以浮窗形式叠加在游戏画面右侧。同时支持世界状态、虚空裂缝、仲裁信息，以及可选的 AutoHotkey 键鼠宏。OCR 与浮窗不读取或修改游戏进程；宏属于自动化输入功能，请使用者自行了解并承担游戏规则与账号风险。

## 功能一览

| 功能     | 说明                                                                                            |
| ------ | --------------------------------------------------------------------------------------------- |
| 遗物查价   | 框选四个奖励区域 → 快捷键触发 → OCR 识别；先在右侧浮窗显示部件名称与“搜索中...”，随后逐项展示 Warframe Market 最低价和均价，单项查询总超时为 7 秒 |
| 游戏信息浮窗 | 夜灵平野、奥布山谷、魔胎之境轮换倒计时 + 虚空商人到达/离开倒计时，30 秒自动刷新                                                   |
| 虚空裂缝浮窗 | 当前全部裂缝任务列表，含九重天风暴裂缝，60 秒自动刷新                                                                  |
| 自动监测   | 每 200ms 监测屏幕左上角，检测到遗物奖励画面自动触发查价，结算完成后浮窗自动关闭                                                   |
| 实时仲裁   | 当前及后续仲裁任务，含节点、任务类型、敌方阵营                                                                       |
| 代理系统   | 直连 / 自动选择 / allorigins.win / 公共代理池 / cors.sh / 自定义（含 SOCKS5 / HTTP CONNECT 认证），cors.sh 免费层计数器 |
| 主题切换   | 主界面与信息浮窗独立配色，支持暗色/亮色等多种预设                                                                     |
| 中英对照表  | 3837 项物品中英文名称对照，支持从 Market API 在线更新，缓存至本地                                                     |
| 键鼠宏    | 可选的 AutoHotkey 键鼠宏，支持录制、手动添加延迟、拖动排序、延迟叠加/删除、标准延迟、按压时长随机、动作间隔随机及多种触发方式                   |
| 浮窗守护   | 世界状态、虚空裂缝和识别结果浮窗自动维护置顶状态，被其他窗口临时遮挡后会自动恢复，同时不抢占游戏输入焦点                                |

## 运行要求

- Windows 10 19041 或更高版本（x64）
- Windows 已安装「简体中文」语言包及 OCR 组件
- Warframe 使用无边框全屏或窗口模式
- 程序为自包含发布（self-contained），无需额外安装 .NET Runtime

## 使用说明

1. 下载 `Hertz-Prime的核桃助手.exe`，放入任意文件夹，双击运行
2. 打开遗物奖励画面，点击「框选范围」，框住四个奖励名称所在整行，按 Enter 保存，框选时请仅框选prime部件名称文字所在的区域，以提高识别率。
3. 默认快捷键 `Ctrl + Alt + Q` 触发查价；在主界面「快捷键」区域可自定义
4. 勾选「游戏信息浮窗」和「虚空裂缝浮窗」开关，实时数据将浮动显示在屏幕上
5. 若直连 Warframe API 不稳定，可在「网络与代理」中切换代理模式
6. 放弃使用时仅需直接删除exe文件即可，无需卸载，不要忘记删除缓存文件，缓存文件位于...\Documents\Hertz-Prime的核桃助手\（...\文档\Hertz-Prime的核桃助手）下，程序最下方也有显示。

识别到奖励名称后，结果浮窗不再等待全部 Market 请求完成：名称会立即出现，价格位置暂时显示“搜索中...”，每个部件查询完成后分别更新最低价和均价；失败或超过 7 秒时显示“从WM搜索失败了🥺”。如果四个奖励全部识别为福马蓝图，普通部件卡片会替换为“四个福马”的金色彩蛋提示。

## 宏功能

宏功能默认作为可选功能提供，通过内置 AutoHotkey v2 运行时向当前前台窗口模拟键盘和鼠标输入：

- 点击「配置宏」可以创建、编辑、启用或删除多个宏
- 支持键盘和鼠标动作录制；录制计时从第一次实际输入开始
- 按下动作在文字下方显示向下三角，抬起动作在文字上方显示向上三角
- 延迟卡片支持直接编辑、添加、删除和拖动排序；多张连续延迟卡片按叠加时间执行
- 延迟大于 1000 毫秒时自动改用小数秒显示
- 删除两个动作之间的显式延迟后，默认使用 50ms 动作间隔
- 可选择统一标准延迟，或分别启用“按压时长随机”和“动作间隔随机”；随机值会在宏每次执行到相应动作时重新生成
- 保存宏时会检查“按下后未抬起”和“抬起但没有对应按下”，发现问题后由用户决定是否仍然保存
- 编辑器录制期间会压暗其他设置区域，并显示醒目的 `● REC 录制中` 标识

首次主动启用宏会显示全屏风险警告。只有点击「我已知晓」后宏才会启用；确认状态会保存到本地，以后不再重复提示。点击「放弃启用」或关闭警告窗口不会启用宏，也不会保存确认状态。

使用宏前请阅读 Warframe 官方资料：

- [第三方软件与您](https://support.warframe.com/hc/zh-cn/articles/360030014351-%E7%AC%AC%E4%B8%89%E6%96%B9%E8%BD%AF%E4%BB%B6%E4%B8%8E%E6%82%A8)
- [Warframe 隐私政策](https://www.warframe.com/zh-hans/privacy)
- [Warframe 使用条款](https://www.warframe.com/en/terms)

## 代理配置

程序内置多种代理方式，用于访问 `api.warframestat.us`：

| 模式             | 说明                                                                                    |
| -------------- | ------------------------------------------------------------------------------------- |
| 直连             | 不走代理，直接访问 API（`api.warframestat.us使用Cloudflare来限制访问，Cloudflare大概率是限制了国内ip，所以大概率无法直连`） |
| 自动选择           | 直连优先，失败后依次尝试 cors.sh → allorigins.win → 公共代理池                                         |
| allorigins.win | 通过 allorigins.win 转发（不一定稳定，经测试作者所在区域无法使用）                                             |
| 公共代理池          | 从多个免费代理列表来源中自动筛选可用节点（经测试很垃圾，非常不稳定）                                                    |
| cors.sh        | 通过 cors.sh 转发，免费层 10,000 次/月，主界面实时显示计数（最稳定）                                           |
| 自定义            | 填入任意代理基础地址，或点「高级」配置 SOCKS5 / HTTP CONNECT 认证                                          |

Warframe Market API 默认直连，可勾选「代理 WM API」纳入代理路由（不建议）。

## 注意事项

- 查价功能读取 Warframe Market 公开只读接口，不登录、不下单、不操作 Warframe 进程
- OCR、世界状态和查价浮窗不会读取、注入或修改 Warframe 游戏进程；宏功能会模拟外部键鼠输入，性质和风险与纯查价功能不同
- Warframe 官方没有提供永久有效的第三方软件白名单。第三方软件和自动化工具是否违规取决于其功能、使用方式及官方当时的政策，使用者需自行承担账号风险
- 不建议使用宏进行无人值守、AFK 刷取、自动瞄准、规避游戏机制或其他脱离玩家控制的自动化行为
- 请退出助手后再关闭电脑，防止下次启动电脑网络异常导致部分网页、程序无法正常使用。
- 使用笔记本电脑时建议插电运行（自动监测会持续消耗约 5%–15% CPU），若运行Warframe吃力，请不要打开此功能。
